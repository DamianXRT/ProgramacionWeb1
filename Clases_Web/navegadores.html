Navegadores Populares

Chrome

Google chrome o mas dicho chrome es el navegador web mas popular ala hora de usar internet, a conseguido esta reputación gracias a todos sus usuarios y buen mantenimiento que hacen los desarrolladores ala hora de optimizar la plataforma. Es de aclarar que chrome es de codigo cerrado, quiere decir que regida por una empresa de desarrollo(Google), fue creado atraves de proyectos de codigo abierto para poder darle forma, estructura, seguridad, fuidez y conectividad.


Ala hora de navegar por la web lo usuarios eligen siempre la opción de Chrome al tener seguridad y estabilidad, pero cabe recalcar que no es el unico que hace muy bien su trabajo.

Año de lanzamiento 2 de septiembre de 2008
